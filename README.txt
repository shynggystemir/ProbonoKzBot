1. Установи Python и библиотеки:
   pip install -r requirements.txt

2. В файле bot.py замени строку:
   ApplicationBuilder().token("PASTE_YOUR_BOT_TOKEN_HERE")
   на свой токен от @BotFather

3. Запусти бот командой:
   python bot.py

4. Он начнёт задавать вопросы и сформирует файл zayavlenie_ali.docx
